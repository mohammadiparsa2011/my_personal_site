# My Personal Site (Django Project)

This repository contains a fully professional Django project developed with strict adherence to Clean Code principles, SOLID design, modular architecture, automated testing, CI/CD pipelines, and a standard enterprise-grade structure.

## Project Goals
- Learn and implement professional software architecture  
- Follow PEP8 standards and Django best practices  
- Build a scalable and testable codebase  
- Get familiar with GitHub Flow and GitHub Actions  
- Practice project management at a corporate level

# سایت شخصی من (پروژه جنگو)

این مخزن مربوط به یک پروژه‌ی جنگوی کاملاً حرفه‌ای است که با رعایت تمام اصول Clean Code، SOLID، معماری ماژولار، تست‌های خودکار، CI/CD، و ساختار استاندارد شرکتی توسعه داده می‌شود.

## اهداف پروژه
- یادگیری و پیاده‌سازی معماری حرفه‌ای  
- رعایت اصول PEP8 و بهترین شیوه‌های Django  
- ایجاد ساختار توسعه‌پذیر و تست‌پذیر  
- آشنایی با GitHub Flow و GitHub Actions  
- تمرین مدیریت پروژه در سطح شرکتی
